SmartServe Frontend (Bootstrap mock)

How to run:

1. unzip this project (smartserve-frontend-bootstrap.zip)
2. cd smartserve-frontend
3. npm install
4. npm run dev

Notes:
- This mock hits http://127.0.0.1:8000 by default. Change URLs in the src/pages/*.jsx if your backend is elsewhere.
- This is a simple test UI for signup/login/report creation + image upload.
